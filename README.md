# Project-07-Multi-Agent-Web-Research-and-Synthesis-Engine
An autonomous AI system utilising interacting agents to scrape, synthesise, and summarise web research. Built with Python, LangChain, and the OpenAI API.

## Overview
The **Multi-Agent Web Research & Synthesis Engine** is a fully autonomous, production-level agentic AI system designed to think, search, read, and write. [cite_start]Moving far beyond simple chatbots or standard QA tools, this system features a team of specialised, interacting AI agents that collaborate to scrape, synthesise, and summarise web research into professional reports on any topic. 

## Architecture: The Agent Team
The system operates on a shared memory architecture that acts as a "unified brain" for our collaborative agents:
* **Reader Agent:** Autonomously searches, ingests, and comprehends complex source material.
* **Writer Agent:** Synthesises the gathered information and drafts a comprehensive, professional research report.
* **Critic Agent:** Reviews, critiques, and refines the drafted report to ensure high-quality, accurate, and polished output.

## Tech Stack
* **Framework:** LangChain, LCEL (LangChain Expression Language)
* **Core Intelligence:** Advanced LLMs
* **System Design:** Multi-Agent Collaboration Pipeline with Shared Memory
